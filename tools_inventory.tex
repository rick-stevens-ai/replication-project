\begin{abstract}
This document catalogs all computational tools, software packages, libraries, databases, and frameworks required to replicate 31 scientific papers identified as candidates for AI-based replication from the OSTI corpus of ${\sim}$67,000 DOE-funded publications.
Papers 1--11 were identified in an initial broad screen; papers 12--31 were selected in a focused search for \textbf{high AI-replicability} candidates that use only open-source tools, are purely computational, and have clear quantitative outcomes.
\end{abstract}

\section{Recommended Environment Setup}
%% ============================================================

\begin{enumerate}[nosep]
\item \textbf{Python environment:} \tool{conda create -n replicate python=3.10}
\item \textbf{Core ML:} \tool{pip install torch tensorflow keras transformers scikit-learn scikit-image}
\item \textbf{Generative AI:} \tool{pip install rdkit-pypi smilesx umap-learn bertopic}
\item \textbf{Scientific:} \tool{pip install numpy scipy matplotlib astropy sympy pandas gpy gpytorch botorch torchsde}
\item \textbf{Domain:} \tool{pip install pyemma healpy corner ray[tune] tensorflow-probability}
\item \textbf{R packages:} \tool{install.packages(c("fldgen","ncdf4","hetGP","DiceOptim","GPareto","mlrMBO","ggplot2","dplyr","tidyr"))}
\item \textbf{Bioinformatics:} \tool{conda install -c bioconda hmmer blast mafft diamond mmseqs2 iqtree prodigal checkm-genome metabat2 bowtie2}
\item \textbf{Compiled codes:} Build LAMMPS, FDTD, ScaWL, GraphBLAS from source
\item \textbf{Licensed codes:} Obtain VASP, SCALE, HFSS/COMSOL, MATLAB as needed
\item \textbf{GPU:} NVIDIA GPU with $\geq$16\,GB VRAM for PyTorch/TensorFlow training
\end{enumerate}
